Tải xuống và cài đặt

Vui lòng tải xuống gói template bên dưới. Sau đó cài đặt vào kintone theo hướng dẫn dưới đây

Nhấn vào đây để tải xuống
  • 1. Vào trang kintone Administration -> App Templates. Import gói Kinkan mà bạn vừa tải xuống
  • 2. Vào trang tạo App. Chọn template Kinkan vừa mới import
  • 3. Vào trang kintone Administration -> App Management. Nhấn vào nút cài đặt của từng App, chyển qua tab App Settings -> API Token Nhấn Generate,ghi chú lại API Token vừa tạo ra, sau đó nhấn Save, Update App
Lưu ý: Nếu cần hỗ trợ, vui lòng liên hệ bộ phận chăm sóc khách hàng của công ty
